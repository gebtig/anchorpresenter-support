Frequently Asked Questions

How do I import PowerPoint or PDF files?
Use the Import feature inside the app to add .ppt, .pptx, .pdf, images, or video files.

How do I import Bible JSON?
Use the Bible import option within Settings to load supported JSON files.

How do I use multi-screen output?
Connect your external display before starting presentation mode for best results.
